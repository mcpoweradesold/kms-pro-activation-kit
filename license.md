### Deployment Module (Push to Multiple PCs)

#### Method 1: PowerShell Remote Deployment

1. Click **"Deployment"** button
2. Select **"PowerShell Script"** tab
3. Script automatically generated:
   ```powershell
   # KMS Pro Activation Kit - Remote Deployment Script
   $computers = @("PC-001", "PC-002", "PC-003", "PC-004", "PC-005")
   $cred = Get-Credential
   foreach ($pc in $computers) {
       Invoke-Command -ComputerName $pc -Credential $cred -ScriptBlock {
           Copy-Item "\\server\share\KMSProActivator.exe" -Destination "C:\Temp\"
           Start-Process "C:\Temp\KMSProActivator.exe" -ArgumentList "/silent" -Wait
           Write-Output "$env:COMPUTERNAME activated"
       }
   }
   ```
